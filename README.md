🎯 Portfolio Shortlisting System
This is a Portfolio Shortlisting System built using Retrieval-Augmented Generation (RAG), LangChain, Groq LLM, and ChromaDB. The system allows job consultancy services to intelligently match resumes to job descriptions using AI and Generative LLMs.

Deploys easily using Streamlit.

📺 Live Demo: Coming Soon (Render Deployment)

🚀 GitHub Repo: https://github.com/sadarerahul/portfolio-shortlisting

📌 Features
✅ Upload resumes in text or file format (PDF/DOCX parsing coming soon)

🔍 Enter a job description and get a list of matching candidates

🤖 Uses Groq LLM via LangChain to compare resume content

📚 Embeds resume text using ChromaDB and retrieves matches using RAG

📩 Generates an automatic email response with matched profiles

🛠️ Tech Stack
Python

Streamlit

LangChain + Groq LLM (via langchain_groq)

ChromaDB

FAISS (optional alternative to ChromaDB)

Streamlit File Upload and Display
